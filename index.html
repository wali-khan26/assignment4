Dataset 1 — Tree Visualization
Animal Kingdom Taxonomy
Biological classification of animals from Kingdom → Phylum → Class → Order → Family → Genus → Species. Source: Encyclopedia of Life / Standard Biological Classification.

Nodes represent taxonomic ranks. Edges represent parent-child relationships between ranks. Ideal for tree visualization: single root (Animalia), no cycles, strict hierarchy.
Nodes
Taxonomic ranks — Kingdom, Phylum, Class, Order, Family, Genus, Species
Edges
Parent → child relationships between adjacent taxonomic levels

Dataset 2 — Graph Visualization
Les Misérables Character Network
Co-appearance network of characters in Victor Hugo's novel Les Misérables. Source: D.E. Knuth, The Stanford GraphBase (1993).

Nodes represent characters. Edges connect characters who appear in the same scene. Edge weight = number of co-appearances. Ideal for graph: has cycles, multiple paths, weighted edges.
Nodes
Characters in the novel — node size = degree centrality
Edges
Co-appearances in same scene — edge thickness = frequency
